Una herramienta web moderna y minimalista para visualizar y formatear JSON.

## ✨ Características

- **Formateo automático** - Pega tu JSON y se formatea instantáneamente
- **Detección de errores** - Avisa si el JSON tiene errores de sintaxis y muestra la línea
- **Resaltado de sintaxis** - Colores para keys, strings, números, booleanos y null
- **Estadísticas** - Muestra cantidad de líneas y tamaño
- **Ejemplo integrado** - Un sample para probar rápidamente
- **Diseño responsive** - Funciona en móvil y escritorio

## 🚀 Uso

Simplemente abre `index.html` en tu navegador. No requiere servidor ni dependencias.

## ⌨️ Atajos

- `Ctrl + Enter` - Formatear
- `Ctrl + Shift + V` en el navegador - Pegar y formatear automáticamente
